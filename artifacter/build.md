# /build
## 概要
原神のビルドカードを生成します。

## 使用方法
```
/build <option>
```

オプション名 | 概要 | 必要かどうか
--- | --- | --
uid / user | 他のユーザーのデータを表示する。 | いいえ
score | スコアの計算式を変更する。 | いいえ

!!!Info scoreオプションの値
scoreオプションには
- 攻撃力換算
- HP換算
- 防御力換算
- 元素チャージ効率換算
- 元素熟知換算
のいずれかが入力可能です。
!!!

## 動作
```mermaid
graph LR
    A{/build}
    A -->|uid / user を指定した場合| B[指定したユーザーのデータを表示]
    A -->|指定していない場合| C{あなたのデータを表示}
    C -->|uidを登録していない場合| D[uid登録フォームを表示]
    C -->|uidが登録されている場合| E[データを表示]
```
