# 音声合成API
## 概要
本APIはVOICEVOX Engineの技術を使用したAPIサービスとなっております。

APIの仕様につきましてはVOICEVOXのドキュメントをご覧ください。

サービスページ: https://voicevox-api-pub.kuroneko6423.com/
