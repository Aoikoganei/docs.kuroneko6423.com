# /skip
## 概要
/skipではBOTが読み上げている最中に止める事ができます

# skip

## 使用方法
```
/skip
又は s とだけ入力
```

## 概要
`/skip`について

/skipは、現在読み上げている内容・今後読み上げる内容(キューに入っているもの)を全てスキップします。
何かを参照する際の長文をスキップしたい場合に、`/skip`又は、`s`とだけ入力すると、会話を終了いたします。
